# MyBudget Budgeting App

A simple Python app that tracks expenses and visualises spending using graphs.

**Overview**

This project is a simple budgeting app where users can log expenses, set budgets and predict expenditure. 
The project is an intermediate exploration of data handling, as well as basic databases and machine learning. 
The project allows users to keep track of their expenses and manage their budgets in a time of rising inflation.

**Features**

- log expenses
- delete expenses
- predict expenditure
- auto categorised inputs
- graphs of expenditure

**Tools**

 - Python 3.11 (Language)
 - Matplotlib.Pyplot (Graphing)
 - Pandas (Data Handling)
 - Scikit-Learn (Machine Learning Prediction and Classification)
 - Os (Path Checking and confirmation)
 - Numpy (Data Structure Handling and Mathematical Operations)
 
  **How To Run**

   Method 1

   1. Clone Repository
   2. Install Dependencies : pip install matplotlib pandas scikit-learn numpy
   3. Run the script : python MyBudget.py

  Method 2

  1. Zip Folder
  2. Extract All
  3. Open in VSC or editor
  4. Run in terminal

  **Reflection**

   This project is one of my early-intermediate projects (2026 May) exploring data handling and machine learning.
   From creating this project I learned how to use multiple linear regression to predict outputs based on previous data,        classification with Naive Bayes and Count Vectorizer, as well as CSV database systems.
   Additionally, I learned to visualise data in different types of graphs such as pie graphs and bar graphs.
   Some limitations of this project is the csv database because it is not very scalable.
   Moreover, the passwords are unencrypted using hash which reduces security.
   Finally, one last limitation is that the auto classifying feature uses a general data source, and is not based on            individual data.
   If I improved this project in the future, I would add a GUI, hashed passwords and an improved SQL database rather than a     csv database.
